# codex-switcher-menubar

This is a small macOS Menu Bar app I built for my own use.

It was made with Vibe Coding.

Credits:

- UI style is inspired by [Claude Usage Bar](https://github.com/Blimp-Labs/claude-usage-bar)
- Account switching and core workflow were learned from [Codex Switcher](https://github.com/Lampese/codex-switcher)
- If you want the fuller original app, please use Codex Switcher directly

Why not just use the original version?

- I wanted a lightweight app that stays in the Menu Bar
- I mainly care about fast account switching and quick usage checks
- I do not need a fuller desktop manager UI for my workflow

So this repo is not meant to replace Codex Switcher. It is just a more minimal Menu Bar interpretation of it.
